# **App Name**: Dilsey

## Core Features:

- Voice Chat Rooms: Users can create and join voice chat rooms on various topics, with options for moderation.
- Profile Creation: Users can create profiles with customizable avatars and bios.
- Friend Connections: Users can send friend requests and connect with other users within the app.
- AI-Powered Recommendations: An AI tool suggests relevant chat rooms based on user interests and profile information.
- Real-time Text Chat: In addition to voice, users can send text messages within chat rooms.
- Gifting and Reactions: Users can send virtual gifts and animated reactions to other users during chats.

## Style Guidelines:

- Primary color: Warm coral (#FF7F50) to evoke feelings of warmth and connection.
- Background color: Light peach (#FAF0E6) to create a soft and inviting atmosphere.
- Accent color: Dusty rose (#D87093) to highlight interactive elements and calls to action.
- Headline font: 'Belleza', a humanist sans-serif, for headlines and short amounts of text. Body font: 'Alegreya', a humanist serif, for a slightly more literary feel.
- Use soft, rounded icons with subtle gradients to complement the warm color palette.
- The layout should prioritize ease of use and clear navigation to promote user engagement.
- Subtle animations and transitions should be used to create a smooth and engaging user experience.
